# Cypress Automation Test – SauceDemo

## Mô tả
Project này sử dụng Cypress để test chức năng website:
https://www.saucedemo.com

## Các test case

### Login Test
- Login thành công với tài khoản hợp lệ
- Hiển thị lỗi khi login sai

### Cart Test
- Thêm sản phẩm vào giỏ
- Sắp xếp sản phẩm theo giá
- Xóa sản phẩm khỏi giỏ
- Checkout thành công

## Cách chạy project

Cài đặt dependency:
npm install

Mở Cypress:
npx cypress open

Run test:
Chọn file test trong giao diện Cypress

## Công nghệ sử dụng
